# Heart Disease Prediction using Logistic Regression

## 📌 Overview
This project predicts whether a person is likely to have heart disease based on medical attributes using a Logistic Regression model.

## 🧠 Model
- Algorithm: Logistic Regression
- Library: Scikit-learn
- Type: Binary Classification

## 📊 Dataset
- Dataset: Heart Disease Dataset
- Features include:
  - Age
  - Sex
  - Chest Pain Type
  - Cholesterol
  - Max Heart Rate
  - etc.
